# Your Store – Web Automation Framework

A complete Web UI Automation Framework for testing the **Your Store e-commerce website (awesomeqa.com)**.

This project demonstrates an end-to-end automation solution using industry standard tools and best practices for functional and regression testing.

---

## Overview

The framework is built using:

- **Java**
- **Selenium WebDriver**
- **TestNG**
- **Maven**
- **Page Object Model (POM)**
- **Extent Reports**

It includes **30+ automated test cases** covering major e-commerce functionalities such as:

- User Registration  
- Login / Logout  
- Product Search  
- Wishlist & Compare List  
- Cart Management  
- Checkout Flow  
- Currency Switching  
- Category Navigation  
- Negative and Edge Case Scenarios  

---

## Key Features

- Clean and scalable **Page Object Model (POM)** design  
- Organized Maven project structure  
- Positive and negative test coverage  
- Randomized category selection and hover functionality  
- Detailed HTML reports using **Extent Reports**  
- Reusable utilities and base components  
- Supports Chrome browser via **WebDriverManager**  
- Easy to execute and maintain
