This is a complete Web UI Automation Framework for testing the Your Store e-commerce website (awesomeqa.com).
The framework is built using Java, Selenium WebDriver, TestNG, Maven, and the Page Object Model (POM).
It includes 30+ automated test cases covering registration, login, search, wishlist, compare list, cart, checkout flow, and more.

Features

Clean & scalable Page Object Model (POM)
Positive & negative test coverage
Randomized category selection & hover tests
Extent Reports integrated for detailed HTML reports
Maven project structure with reusable components
Supports Chrome via WebDriverManager
